## 📋 Project Overview

This is a relational database project built in MySQL. The system tracks game players performance across different game levels. If a player scores low on a level, the system detects the weak area and assigns a training mission to help them improve.

## 🗄️ Database Tables

| Table | Purpose |
|-------|---------|
| `players` | Player name, email, skill level, join date |
| `game_levels` | Game levels with difficulty and topic |
| `player_scores` | Each player score on each level |
| `training_missions` | Missions assigned to weak players |

---

## ⚙️ How to Run

1. Open **MySQL Workbench**
2. Go to **File → Open SQL Script**
3. Select `team_nisar_gaming.sql`
4. Press `Ctrl + Shift + Enter` to run
5. Done — database is ready!
